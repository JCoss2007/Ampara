# Ampara
Created with Visual Studio Code
Ampara es una plataforma digital enfocada en brindar apoyo emocional, orientación psicológica y herramientas de seguridad para mujeres que enfrentan situaciones de estrés, ansiedad o posibles episodios de violencia familiar.

## Segmentos Objetivo
- Mujeres de 18 a 40 años que requieren orientación emocional, acompañamiento seguro o herramientas de emergencia.

- Psicólogas profesionales encargadas de brindar soporte emocional, seguimiento y asesoría dentro de un entorno digital seguro.

## Principales Características
- Botón SOS para alertar a contactos de emergencia previamente registrados.
- Chat seguro entre usuaria y psicóloga.
- Recursos informativos sobre bienestar, prevención y autocuidado.
- Panel profesional para la gestión de casos y seguimiento.

## Tecnologías Utilizadas
- Visual Studio Code (editor principal de desarrollo)
- HTML5, CSS3 y JavaScript
- Figma para el diseño y prototipado de interfaces
- GitHub para control de versiones y repositorio del proyecto
- GitHub Pages para despliegue del prototipo web
